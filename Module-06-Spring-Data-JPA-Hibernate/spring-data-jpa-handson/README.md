## Difference

### JPA

Java Persistence API is a specification for ORM.

### Hibernate

Hibernate is the most popular implementation of JPA.

### Spring Data JPA

Spring Data JPA builds on top of JPA and Hibernate, reducing boilerplate code by providing repositories like JpaRepository.
