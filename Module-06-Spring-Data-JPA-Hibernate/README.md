# Module 6: Spring Data JPA and Hibernate

This module focuses on object-relational mapping (ORM) and database interactions using Spring Data JPA and Hibernate.

## Hands-on Projects

1. **spring-data-jpa-handson**
   - Setup of basic Spring Data JPA using `JpaRepository`.
   - Domain entity modeling and simple CRUD operations (e.g., managing `Country` data).

2. **spring-data-jpa-handson-2**
   - Advanced entity relationships (`Department`, `Employee`, `Skill`).
   - Creating custom finder methods in repositories (`findByName`, `findByDepartmentName`).
   - Testing custom repository queries.

3. **spring-data-jpa-handson-3**
   - Writing custom complex queries using the `@Query` annotation.
   - Implementation of both HQL (Hibernate Query Language) and Native SQL queries.

## Tech Stack
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 In-Memory Database / MySQL
