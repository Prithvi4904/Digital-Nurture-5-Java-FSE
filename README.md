# Digital Nurture 5 - Java Full Stack Engineering (FSE) 🚀

Welcome to my Java Full Stack Engineering training repository! This repository contains all my coursework, exercises, and projects completed during the Digital Nurture 5 program. 

The repository is organized into modular folders, each focusing on a core competency required for a modern Java Full Stack Developer.

## 📁 Modules Covered

*   **Module 1: Design Patterns**
    *   Implementation of common software design patterns (Creational, Structural, Behavioral) using Java.
*   **Module 2: Data Structures and Algorithms**
    *   Core computer science concepts, algorithm implementation, and problem-solving using Java.
*   **Module 3: PL/SQL**
    *   Database management, complex querying, stored procedures, and triggers.
*   **Module 4: TDD, JUnit & Mockito**
    *   Test-Driven Development practices, unit testing with JUnit 5, and dependency mocking with Mockito.
*   **Module 5: Spring Core & Maven**
    *   Inversion of Control (IoC), Dependency Injection (DI), Maven project configuration, and Spring XML contexts.
*   **Module 6: Spring Data JPA & Hibernate**
    *   Entity mapping, Repositories, custom queries (HQL & Native SQL), and ORM principles.

## 🛠️ Tech Stack
*   **Language:** Java (JDK 8 / 17)
*   **Build Tool:** Maven
*   **Frameworks:** Spring Core, Spring WebMVC, Spring Data JPA, Hibernate
*   **Testing:** JUnit 5, Mockito
*   **Database:** SQL / PL/SQL

## 💡 About this Repository
This repository serves as a portfolio of my backend engineering skills. Every module is structured cleanly with its own documentation, adhering to industry best practices such as modular design, dependency injection, and proper version control management.
