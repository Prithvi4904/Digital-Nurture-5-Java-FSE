# Module 03 - PL/SQL

## Overview

This module contains PL/SQL exercises focused on database programming concepts used in real enterprise applications.

## Exercises

| Exercise | Topic | Description |
| --- | --- | --- |
| `Exercise-01-ControlStructures` | Control Structures | Uses cursors, conditional statements, loops, updates, and `DBMS_OUTPUT` |
| `Exercise-03-StoredProcedures` | Stored Procedures | Implements reusable procedures for banking and employee operations |

## Tools Required

- Oracle Database
- SQL*Plus, SQL Developer, or any Oracle-compatible SQL execution environment

## How to Execute

Enable server output before running scripts:

```sql
SET SERVEROUTPUT ON;
```

Then run each `.sql` file in the required exercise folder.

## Submission Notes

Each exercise folder contains:

- Individual `.sql` solution files
- A README describing the objective, scripts, logic, and expected behavior
