# Spring REST Handson 1

## Objective

Create a Spring Boot REST application that loads Country data from a Spring XML configuration.

## Technologies Used

- Java 17
- Spring Boot 3.5.4
- Spring Web
- Spring Core
- Maven

## Features

- XML Bean Configuration
- REST Controller
- Service Layer
- DAO Layer
- Exception Handling

## Endpoint

GET /country

## Response

```json
{
  "code": "IN",
  "name": "India"
}
```
